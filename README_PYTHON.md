# 物理实验数据采集软件 - Python 图形界面

## 软件简介

这是一个基于 PyQt6 的物理实验数据采集软件，采用 Win11 风格的现代化界面设计。软件支持多种传感器模块的数据采集和管理，目前实现了超声波位移测量模块。

## 功能特性

### 界面设计
- **Win11 风格**：现代化的界面设计，符合 Windows 11 设计语言
- **侧边栏导航**：支持多个传感器模块的快速切换
- **响应式布局**：自适应窗口大小，提供良好的用户体验

### 超声波位移模块
- **实时数据采集**：通过串口实时接收超声波传感器数据
- **数据可视化**：实时显示距离-时间曲线图
- **数据记录**：显示详细的数据记录和统计信息
- **数据导出**：支持将数据保存为 CSV 格式文件
- **参数配置**：可调节采样率、选择串口等参数

### 扩展性
- **模块化设计**：便于添加新的传感器模块
- **统一接口**：所有模块使用相同的界面框架
- **预留接口**：已预留温度传感器、光电门、力传感器等模块位置

## 安装说明

### 环境要求
- Python 3.8 或更高版本
- Windows 10/11 操作系统（推荐）

1. **安装 Python 依赖包**：

2. **运行软件**：
   ```bash
   python run.py
   ```


- **pyserial**：串口通信
- **matplotlib**：数据可视化图表

## 使用说明

### 连接超声波传感器

1. **硬件连接**：
   - 将 WeMOS D1 开发板通过 USB 连接到电脑
   - 按照硬件连接图连接 HC-SR04 模块

2. **软件操作**：
   - 运行软件，选择"超声波位移"模块
   - 在控制面板中选择正确的串口
   - 点击"连接"按钮建立串口连接
   - 点击"开始采集"开始数据采集

### 数据采集流程

1. **连接设备**：选择串口并连接
2. **开始采集**：点击"开始采集"按钮
3. **实时观察**：在图表和文本区域观察实时数据
4. **停止采集**：点击"停止采集"结束数据采集
5. **保存数据**：点击"保存数据"将数据导出为 CSV 文件
6. **清除数据**：点击"清除数据"清空当前数据

### 数据格式

软件接收的数据格式为：`时间戳(us),回波时间(us)`
- 示例：`123456,1450` 表示在 123.456 毫秒时，回波时间为 1450 微秒
- 距离计算公式：`距离(cm) = 回波时间(us) / 58.0`

## 扩展开发

### 添加新模块

要添加新的传感器模块，需要：

1. 创建新的模块类，继承自 `QWidget`
2. 在 `MainWindow` 类的 `init_ui` 方法中添加模块到内容堆栈
3. 在侧边栏添加对应的模块项

### 示例代码结构

```python
class NewSensorWidget(QWidget):
    def __init__(self):
        super().__init__()
        self.init_ui()
    
    def init_ui(self):
        # 创建界面布局和控件
        pass

# 在主窗口中添加
new_module = NewSensorWidget()
self.content_stack.addWidget(new_module)
self.modules["新模块名称"] = new_module
```

## 故障排除

### 常见问题

1. **无法找到串口**：
   - 检查设备管理器中的串口设备
   - 确保驱动程序已正确安装
   - 尝试重新插拔 USB 线

2. **数据接收异常**：
   - 检查波特率设置（应为 115200）
   - 验证硬件连接是否正确
   - 检查传感器模块是否正常工作

3. **图表显示异常**：
   - 确保 matplotlib 正确安装
   - 检查数据格式是否符合要求

### 调试模式

如需调试，可以修改代码中的日志输出或使用 Python 的调试工具。

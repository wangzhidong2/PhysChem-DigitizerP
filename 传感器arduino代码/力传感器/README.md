# HX711 力/质量传感器模块

## 概述

本模块基于 HX711 24位高精度 ADC 芯片和称重传感器，实现力/质量的实时测量。通过 ESP32-S3 采集数据，支持有线串口和 BLE 蓝牙两种连接方式，配合上位机 Python 程序实现实时数据显示、去皮、校准和 CSV 导出功能。

## 硬件组成

| 组件 | 说明 |
|------|------|
| HX711 模块 | 24位高精度 ADC，增益 128（通道A）/ 32（通道B） |
| 称重传感器 | 电阻应变片式，推荐量程 1kg~10kg |
| ESP32-S3 | 主控芯片，负责 ADC 读取和数据传输 |

## 接线说明

### ESP32-S3 ↔ HX711 模块

```
ESP32-S3          HX711 模块
────────          ──────────
3.3V      ───→    VCC
GND       ───→    GND
GPIO4     ───→    DT (DOUT)
GPIO5     ───→    SCK (PD_SCK)
```

### HX711 模块 ↔ 称重传感器

```
HX711 模块        称重传感器
─────────        ──────────
E+        ───→   激励正极（红色线）
E-        ───→   激励负极（黑色线）
A+        ───→   通道A正极（绿色线，默认使用）
A-        ───→   通道A负极（白色线）
B+        ───→   通道B正极（未使用）
B-        ───→   通道B负极（未使用）
```

> **注意**：称重传感器线序因厂家不同可能有所差异，请以传感器标注为准。常见线序为：红=E+、黑=E-、绿=A+、白=A-。

## 上位机使用说明

### 连接方式

程序支持两种连接方式：

1. **有线串口**：USB 连接 ESP32-S3，选择对应 COM 口
2. **BLE 蓝牙**：搜索并连接 BLE 设备（需安装 `bleak` 库）

### 去皮操作

1. 连接传感器后，确保称重平台上无负载
2. 点击 **"去皮（TARE）"** 按钮
3. 程序发送 `TARE` 命令，ESP32-S3 重新计算偏移量
4. 收到 `TARE_DONE` 响应后，当前读数归零

### 校准操作

校准用于建立 ADC 原始值与实际质量（克）之间的换算关系。

**步骤：**

1. 点击 **"校准（CALIBRATE）"** 按钮
2. 确保称重平台空载，点击 **"1. 请空载，点击记录零点"**
3. 在弹出对话框中输入已知砝码质量（如 100g），点击确定
4. 将砝码放置在称重平台上，等待读数稳定
5. 点击 **"2. 已放XXXg砝码，点击记录"**
6. 程序自动计算校准比例并保存

**校准原理：**

```
校准比例 (scale) = 已知质量 / (加载ADC - 空载ADC)
偏移量 (offset) = 空载ADC
实际质量 = (当前ADC - offset) × scale
```

**示例：**

```
空载 ADC = -58720
加载 100g 砝码后 ADC = -52720
ADC 差值 = -52720 - (-58720) = 6000
scale = 100 / 6000 = 0.016667
offset = -58720

当 ADC = -55720 时：
质量 = (-55720 - (-58720)) × 0.016667 = 3000 × 0.016667 = 50.0g
```

> 校准参数自动保存到 `force_sensor_config.json`，下次启动时自动加载。

### 数据采集与保存

1. 点击 **"开始采集"** 开始记录数据
2. 实时查看质量值、ADC 原始值和统计信息
3. 点击 **"停止采集"** 结束记录
4. 点击 **"保存数据"** 导出 CSV 文件

**CSV 格式：**

```csv
time_s,raw_adc,force_g
0.000,-58720,0.00
0.080,-58695,0.42
0.160,-58701,0.32
```

## 技术参数

| 参数 | 值 |
|------|----|
| ADC 分辨率 | 24 位 |
| 有效精度 | 约 20 位（取决于传感器） |
| 采样速率 | 10Hz 或 80Hz（由 HX711 速率引脚决定，默认 10Hz） |
| 通道 A 增益 | 128 |
| 通道 B 增益 | 32 |
| 输入电压范围 | ±20mV（通道 A，增益 128） |
| 工作电压 | 2.7V ~ 5.5V |
| 工作电流 | 约 1.5mA |

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| ADC 值始终为 0 | 检查 DT/SCK 接线是否正确 |
| ADC 值跳动很大 | 检查传感器接线是否牢固；避免在气流/振动环境中使用 |
| 校准后质量不准确 | 重新校准；确保空载时稳定；使用更重的砝码校准 |
| 串口连接失败 | 检查 COM 口是否正确；确认 ESP32-S3 已上传代码 |
| BLE 连接失败 | 先在 Windows 蓝牙设置中配对设备 |
| 读数为负数 | 称重传感器 A+/A- 接反，交换即可 |

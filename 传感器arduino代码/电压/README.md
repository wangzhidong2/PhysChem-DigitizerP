# 电压传感器模块使用说明

## 🔗 模块介绍

本模块基于 ESP32-S3 内置 ADC（模数转换器）实现电压测量功能，可直接采集 0-3.3V 范围内的模拟电压信号。适用于物理实验中的电压监测、电池电压检测、传感器输出电压采集等场景。

## 🔌 接线指南

### ESP32-S3 电压采集接线方式

```
ESP32-S3 开发板       外部电压源/传感器
──────────────        ──────────────────
   GPIO1         ←      信号输出（0-3.3V）
   GND           ←      GND（共地）
```

**固件代码**: [ESP32_ADC_Raw_Data.ino](ESP32_ADC_Raw_Data.ino)

**引脚定义**（来自固件）：
```cpp
#define ADC_PIN 1        // 使用 GPIO1 (ADC1_CH0)，ESP32-S3 推荐 ADC 引脚
#define SAMPLE_INTERVAL 100  // 采样间隔 100ms (10Hz)
```

### ⚠️ 接线注意事项

1. **电压范围**：被测电压必须在 **0-3.3V** 范围内，超过此范围可能损坏 ADC！
2. **共地连接**：务必将 ESP32-S3 的 GND 与被测电路的 GND 相连，否则读数无效
3. **引脚选择**：GPIO1 对应 ADC1_CH0，是 ESP32-S3 推荐使用的 ADC 通道
4. **阻抗匹配**：被测信号源输出阻抗建议 < 10kΩ，以保证采样精度
5. **高压保护**：若需测量更高电压，请外接分压电阻网络降压后再接入

### 分压电阻选型示例（测量 0-12V）

若需测量超过 3.3V 的电压，可使用电阻分压：

```
Vin (0-12V) ──┬── R1 (27kΩ) ──┬── Vout (→ GPIO1)
               │                │
              R2 (10kΩ)       GND
               │
              GND
```

分压比 = R2 / (R1 + R2) = 10 / (27 + 10) ≈ 0.270

实际电压 = ADC读取值 / 4095 × 3.3V / 分压比

## 💻 固件说明

### 数据输出格式

**串口输出**：`时间戳(ms),ADC原始值`

**示例数据**：
```
1234,2048
1334,2051
1434,2046
```

**ADC 配置参数**：

| 参数 | 值 | 说明 |
|------|-----|------|
| ADC 引脚 | GPIO1 (ADC1_CH0) | ESP32-S3 推荐 ADC 通道 |
| 分辨率 | 12 位 | 量化范围 0-4095 |
| 衰减 | 11dB | 量程 0-3.3V |
| 采样率 | 10Hz | 100ms 间隔 |
| 波特率 | 115200 | 串口通信速率 |

### 固件核心代码

```cpp
#define ADC_PIN 1        // GPIO1 (ADC1_CH0)
#define SAMPLE_INTERVAL 100  // 100ms 采样间隔
#define ADC_WIDTH_BIT ADC_WIDTH_BIT_12  // 12位精度

void setup() {
  Serial.begin(115200);
  analogReadResolution(12);           // 12位精度
  analogSetAttenuation(ADC_ATTEN_DB_11); // 11dB衰减(0-3.3V)
  pinMode(ADC_PIN, INPUT);
  Serial.println("START");
}

void loop() {
  if (millis() - lastSampleTime >= SAMPLE_INTERVAL) {
    int adcValue = analogRead(ADC_PIN);
    Serial.print(millis());
    Serial.print(",");
    Serial.println(adcValue);
  }
}
```

### ADC 衰减选项参考

| 衰减值 | 量程 | 适用场景 |
|--------|------|----------|
| 0dB | ~0-0.8V | 低电压精密测量 |
| 2.5dB | ~0-1.1V | 一般低电压 |
| 6dB | ~0-1.35V | 中等电压 |
| 11dB | ~0-3.3V | 标准电压测量（本固件采用） |

## 🎯 使用说明

### 上位机功能

1. **实时数据显示**：显示当前 ADC 原始值和换算后的电压值
2. **数据曲线**：实时绘制电压变化曲线
3. **统计信息**：显示平均值、最大值、最小值、标准差
4. **数据保存**：保存为 CSV 格式（包含时间、ADC 值、电压值）
5. **可配置采样率**：支持调整采样频率（默认 10Hz）
6. **校准功能**：支持偏移校准和增益校准

### 操作流程

1. **烧录固件**：使用 Arduino IDE 将 `ESP32_ADC_Raw_Data.ino` 烧录至 ESP32-S3
2. **连接硬件**：按照接线指南连接待测电压源
3. **打开上位机程序**：运行 `main.py`，选择电压传感器模块
4. **连接串口**：选择正确的 COM 口并连接
5. **开始采集数据**：点击"开始采集"按钮
6. **保存数据**：点击"保存数据"，导出为 CSV 文件

### 电压换算公式

```
实测电压(V) = ADC原始值 × (3.3 / 4095)
```

**示例**：
- ADC = 0 → 0V
- ADC = 2048 → 1.65V
- ADC = 4095 → 3.3V

## ⚠️ 常见问题

### Q1: ADC 读数跳动很大
**原因**：信号干扰、接地不良、未加滤波电容
**解决**：
- 检查接地是否良好（必须与被测电路共地）
- 在信号输入端并联一个 0.1μF 电容到地进行滤波
- 远离高频干扰源（电机、开关电源等）
- 在软件中启用多点平均滤波

### Q2: 测量值偏差大
**原因**：ADC 非线性误差、参考电压漂移
**解决**：
- 使用两点校准法：分别测量已知电压（如 0V 和 3.3V），建立线性校正曲线
- 避免在 ADC 量程两端（<5% 或 >95%）进行精密测量
- 多次采样取平均值以降低噪声影响

### Q3: ADC 值异常（始终为 0 或 4095）
**原因**：接线错误、引脚短路或开路、超出量程
**解决**：
- 检查信号线是否正确连接到 GPIO1
- 用万用表测量输入引脚电压是否在 0-3.3V 范围内
- 确认没有短路到 VCC 或 GND

### Q4: 测量精度不够
**原因**：12 位 ADC 固有分辨率限制
**解决**：
- 采用过采样技术（多次采样取平均）
- 使用外部高精度 ADC 模块（如 ADS1115，16 位分辨率）
- 缩小测量范围以提高有效分辨率

### Q5: 找不到串口
**原因**：驱动未安装、USB 线问题、端口被占用
**解决**：
- 安装 CH340/CP2102 驱动
- 更换 USB 数据线（部分充电线无数据传输功能）
- 关闭 Arduino IDE 等可能占用串口的程序

## 🔧 扩展建议

### 提升测量精度

1. **过采样算法**：对连续 N 次采样取平均值，可将有效分辨率提升约 log₂(N) 位
2. **软件滤波**：实现移动平均滤波或卡尔曼滤波抑制噪声
3. **硬件滤波**：添加 RC 低通滤波器（推荐 R=1kΩ, C=0.1μF, fc≈1.6kHz）

### 扩展测量范围

1. **分压电阻网络**：通过精密电阻分压扩展至 0-12V、0-24V 等
2. **运放衰减器**：使用运算放大器构建精确的电压衰减电路
3. **差分测量**：使用仪表放大器（如 INA219）实现差分电压测量

### 多通道采集

ESP32-S3 内置多个 ADC 通道，可同时采集多路电压信号：
- ADC1 通道：GPIO1-GPIO10（共 10 个通道）
- 可用于电池监测 + 传感器输出等多路同步采集

### 无线传输

利用 ESP32-S3 的 WiFi/BLE 功能，实现无线电压监测：
- 远程设备状态监控
- IoT 电压数据上传云端
- 手机 APP 实时查看
